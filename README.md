# Frank 的八年级英语预习资料包（外研版 2024 新课标 · 八年级上册）

结构化学习资料，配合 AI 口语老师（ChatGPT / GPT Live）使用。基于外研版（2024 新课标）英语八年级上册整理，共 6 个 Unit。

## 给 AI 老师的使用说明

1. **先读 `student-profile.md`** — 了解学生 Frank 的情况和教学要求（口语优先 + 美国生活场景）
2. **再读 `toc.md`** — 全册结构、语法主线、每单元阅读与产出任务
3. 按单元教学时，从 `modules.md` 找到对应 Unit 的：学习主题 / 语法中文讲解 / 核心句型 / 单词表 / 课文大意 / 口语问答 / 预习任务
4. 口语课直接取 `speaking_tasks.md` 里对应 Unit 的任务（10 问答 + 5 情景对话 + 3 个一分钟表达 + 10 造句）
5. 检测掌握情况用 `exercises.md`（课本练习清单，答案留空，可让学生口头作答）
6. 单词专项用 `vocabulary.csv`（257 词条：单词/词性/中文/课本例句/口语化例句）

> 课文原文请以学生上传的课本 PDF 为准；`modules.md` 中的课文大意含每篇课文的段落级中文详述和课本页码，可据此定位原文。

## 文件清单

| 文件 | 内容 |
|---|---|
| `student-profile.md` | 学生学习档案 + 教学要求 + 单元×生活场景对照 |
| `toc.md` | 全册目录、单元总表、语法主线、附录页码地图 |
| `modules.md` | 6 个 Unit 完整学习资料（合并版） |
| `vocabulary.csv` | 全册 257 词条（7 列，带表头） |
| `exercises.md` | 课本练习汇总（按 Unit，答案留空） |
| `speaking_tasks.md` | 口语任务包（按 Unit） |
| `extraction_errors.md` | 提取质量记录（0 失败页） |
| `units/` | 分单元原始文件（单元讲解 / 词汇 / 练习 / 口语各自独立） |

2. **动词不定式**：作宾语(+-ing) → 作宾语补足语 → 作目的状语（Unit 3-5）
3. **过去进行时**（Unit 6）
