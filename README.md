# Frank 的八年级英语预习资料包（外研版 2024 新课标 · 八年级上册）

结构化学习资料 + 预习计划，配合 AI 口语老师（ChatGPT / GPT Live）和线下补习课使用。基于外研版（2024 新课标）英语八年级上册整理，共 6 个 Unit。

## 学习模式（重要，AI 老师必读）

Frank **每周五、周日有线下英语补习课**。自学的目标**不是完全学会**，而是：
1. 补习课前把内容**预习完**（预习级达标即可，不抠死）
2. 把预习中不懂的**整理成问题清单**，周五补习课答疑、周日补习课巩固+测试
3. 口语练习必须结合**美国生活场景**（Frank 计划高中毕业后赴美）

## 给 AI 老师的使用说明（按顺序读）

1. **`student-profile.md`** — 学生情况和教学要求（口语优先 + 美国生活场景）
2. **`plans/unit1-emergency-plan.md`** — 本周 Unit 1 急救预习计划（7/16 周四 - 7/18 周六，逐时段任务表）
3. **`plans/six-week-plan.md`** — 后续 6 周计划（7/20-8/30，每周一单元：周一至周四预习 / 周五补习答疑 / 周六只复习 / 周日补习巩固测试）
4. **`plans/tutor-questions.md`** — 问题清单（预填高频疑问 + 空白模板；每天预习后提醒 Frank 记问题，周四打包给补习老师）
5. **`toc.md`** — 全册结构、语法主线、页码地图
6. 每天上课时按计划表定位当天任务，再从下面取内容：
   - 单元讲解/语法/课文大意 → `modules.md`
   - 口语任务 → `speaking_tasks.md`（10 问答 + 5 情景对话 + 3 个一分钟表达 + 10 造句 / 单元）
   - 单词 → `vocabulary.csv`（257 词条：单词/词性/中文/课本例句/口语化例句）
   - 练习检测 → `exercises.md`（答案留空，判卷交给补习老师）

> 课文原文请以学生上传的课本 PDF 为准；`modules.md` 中的课文大意含每篇课文的段落级中文详述和课本页码，可据此定位原文。
