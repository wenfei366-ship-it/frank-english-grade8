# Frank 的八年级英语预习资料包（外研版 2024 新课标 · 八年级上册）

结构化学习资料 + 预习计划，配合 AI 口语老师（ChatGPT / GPT Live）和线下补习课使用。基于外研版（2024 新课标）英语八年级上册整理，共 6 个 Unit。

## 学习模式（重要，AI 老师必读）

**三条铁律：学习顺序严格 · 日历日期灵活 · 补习课节点严格。**

Frank **每周五、周日有线下英语补习课**。自学的目标**不是完全学会**，而是：
1. 补习课前把内容**预习完**（预习级达标即可，不抠死）
2. 把预习中不懂的**整理成问题清单**（周五答疑、周日巩固+测试）
3. 口语练习必须结合**美国生活场景**，Frank 说话时间 ≥70%

**Session 制推进**：内容按 `plans/daily/session-01.md` ~ `session-30.md` 顺序学（每单元 5 次），**不按星期几推算进度**。中断后从 `progress/current-session.md` 记录的位置继续——不补双倍任务，也不因缺课跳过任务。

**词汇三级制**（分级表在 `plans/vocab-abc/`）：A级主动词汇每次 6-8 个（会读会说会造句）/ B级看懂听懂 / C级课文中认识即可。**不要求短期主动掌握整单元全部词汇。**

## 每次学习的标准流程（AI 老师照此执行）

**开课前，按顺序读 3 个文件：**
1. `student-profile.md`（学生情况和教学要求）
2. `progress/current-session.md`（当前进度——确定今天学哪个 Session）
3. 对应的 `plans/daily/session-NN.md`（今天的具体任务）

**上课**：按**授课式教学流程**走（详见 `plans/daily/session-rules.md`，这是教法的唯一标准）：
> ①旧内容复习 10-15 分钟 → ②**单词表带读练读** 15 分钟（先分清 A/B/C 级，逐词带读正音，A 级加练造句）→ ③**课文预习：领读→提问→讲解重点** 25-40 分钟 → ④实际场景口语 20 分钟（Frank 说话≥70%）→ ⑤问题与错误记录 5 分钟
>
> 关键：**先单词后课文**——单词带读过了，读课文才不卡壳；课文要"领读、提问、讲解"三个动作齐全，不是让学生自己闷头看。

**下课后，更新进度：**
- `progress/current-session.md` 打卡 + 更新"下一个 Session"
- 新学 A 级词 → `progress/vocabulary-review.md`
- 口语情况 → `progress/speaking-record.md`
- 说错做错的 → `progress/mistake-bank.md`
- **不依据缺课日期自动跳过任务**

**补习课节点：**
- **周五有课**：整理问题清单（`plans/tutor-questions.md`）带去答疑
- **周五无课**：AI 做复习 + 模拟答疑，问题保留到周日问老师
- **周日课后**：老师反馈 → `progress/tutor-feedback.md`；错题 → `progress/mistake-bank.md`
