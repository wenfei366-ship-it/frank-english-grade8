# 6 周预习计划总览（参考节奏：7月20日 - 8月30日，每周约一个单元）

> **三条铁律**：**学习顺序严格 · 日历日期灵活 · 补习课节点严格**
> - **顺序严格**：内容按 `plans/daily/session-01.md` ~ `session-30.md` 依次推进（每单元 5 次：主课文→语法→听说读写→产出+问题打包★→复习），不跳不乱
> - **日期灵活**：下表日期只是**参考节奏**。哪天中断了，下次打开 `progress/current-session.md` 从记录的位置继续——**不补双倍任务，也不因缺课跳过任何 Session**
> - **节点严格**：每单元第 4 次 Session（问题清单打包★）要赶在**周五补习课之前**完成；实在来不及，优先补"关键课文 + 问题清单"两样
>
> **定位**：自学 = 补习课前预习完 + 整理问题给老师，**不追求自己全学会**。
> **每周补习课节点**（这个是死的）：
> - **周五**：两种模式——
>   - **有补习课**：整理好本周问题清单（`tutor-questions.md`）带去**答疑**
>   - **无补习课**：AI 做**复习 + 模拟答疑**（AI 扮演老师过一遍问题清单，能讲清的讲清，讲不透的保留），问题**保留到周日**问老师
> - **周六**：默认**复习**；如果补习课前的预习还没完成，可以先补齐**关键课文和问题清单**，再复习
> - **周日**：补习课**巩固 + 测试**。课后必做：老师反馈写入 `progress/tutor-feedback.md`，错题写入 `progress/mistake-bank.md`，未解决问题转入下一轮问题清单
> **口语要求**：每次口语环节 Frank 说话时间 ≥70%，必须结合**美国生活场景**（学校生活/寄宿家庭/交朋友/课堂提问/购物/交通/就医/通信/紧急求助）。
> **注**：本周（7/16-7/18）Unit 1 已按急救计划抢跑（见 `unit1-emergency-plan.md`），session-01~05 对 Unit 1 做完整二轮，之后每单元 5 个 session 推进。

| Session | 学什么 | 备注 |
|---|---|---|
| 第1次 | Starting out 导入 + **主课文**（Understanding ideas）+ A级词 6-8 个 | |
| 第2次 | **语法**（Grammar 页 + 中文讲解）+ A级词 6-8 个 | |
| 第3次 | **听说板块** + **读写课文**（Reading for writing）+ A级词 6-8 个 | |
| 第4次 | Presenting ideas 产出任务 + 整单元串讲 + **打包问题清单** | ★补习课硬节点：完成=可去答疑 |
| 第5次 | 复习专场：A级词过筛 / 错题回顾 / 口语串讲 / 语法口头自测 | 不学新内容；预习没完成先补关键课文 |

**每个 Session 的固定五段**（75-95 分钟）：
① 旧内容复习 10-15 分钟 → ② 课本预习 25-40 分钟 → ③ A级词汇 15 分钟 → ④ 实际场景口语 20 分钟（Frank 说话 ≥70%）→ ⑤ 问题与错误记录 5 分钟

**词汇三级制**（分级表见 `plans/vocab-abc/`）：
- **A级主动词汇**：每次 6-8 个，必须会读、会说、会造句
- **B级理解词汇**：能看懂、听懂即可
- **C级识别词汇**：在课文中认识即可
- **不要求短期内主动掌握整单元全部词汇**

## 给 AI 老师的执行要点

1. 每次开课先读 `progress/current-session.md` 定位"下一个 Session"，再打开对应 `plans/daily/session-NN.md` 上课——**不按日期推算该学什么**
2. 口语环节 Frank 说的时间要占 70% 以上；他卡住时先给英文提示词，不行再中文；口语情况记入 `progress/speaking-record.md`
3. 每次结束前 4 件事：①口头小结今天学了啥（让 Frank 说）②疑问记进问题清单、错误记进 `progress/mistake-bank.md` ③更新 `progress/current-session.md` 打卡 ④预告下一个 Session 内容
4. 周六默认复习；**如果补习课前预习尚未完成，可以先补齐关键课文和问题清单**，再复习。Frank 状态好想多学新内容时，以"完成当前 Session + 不透支明天"为度
5. 测试类任务放周日补习课，AI 不代替补习老师判卷；但复习专场可做口头自测热身
6. 周日补习课后必做：老师反馈 → `progress/tutor-feedback.md`；错题 → `progress/mistake-bank.md`；未解决问题保留到下一轮问题清单
7. 暑假结束（约 8/30）时全册 6 单元完成两件事：每单元都预习过 + 每单元都有"已解决"的问题清单——开学即领先
