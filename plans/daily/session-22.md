## ② 课本预习（25-40分钟）

**内容：Grammar 动词不定式作目的状语（PDF p76-77 + `units/unit5.md` 语法讲解 + modules.md Unit 5 语法节）**

1. **先读中文讲解**（`units/unit5.md`"核心语法"节，约10分钟）,抓住三件事：
   - "to + 动词原形"回答 **Why（为什么做）**：I got on the train **to go** home.
   - 三个"马甲"等价：to do = in order to do = so as to do（正式程度递增）
   - 否定：in order **not to** / so as **not to** do（We went on tiptoe so as not to disturb anyone.）
2. **回课文找证据**（约8分钟）：在主课文里找出至少 4 句含 to / in order to / so as to 的句子，口头翻译并指出"目的"部分（参考：to go home / in order to read / in order to block the noise / so as to reduce the smell / so as to see the other passengers' faces）。
3. **做课本练习（口头，约12分钟）**：PDF p76-77 的练习，重点做**合并句子**题（练习6 套路：People wear seat belts in cars. + They want to stay safe. → People wear seat belts **to stay safe** on the road.）。答案说给 AI 听，AI 只指出"对/不对"，讲不清的原理记入问题清单留给补习老师。
4. **易错点自查**（3分钟，口头判断对错）：
   - So as to catch the train, she left early. （✗ so as to 不放句首）
   - He saves money in order to buys a computer. （✗ 不定式后接原形）
   - I closed the door so as don't wake the baby. （✗ 应为 so as not to）

**达标线（预习级）**：能口头合并 2 个句子成目的状语句 + 说出三个"马甲"和否定形式。辨析"作宾语/宾补/目的状语"分不清没关系，记进问题清单。

## ④ 实际场景口语（20分钟）

**场景：在美国坐公交地铁、问路（交通）** —— Frank 说话占比 ≥70%，强制使用今天 A 词（station / seat belt / pick up）+ **今天的语法：每个回合至少说一句带 to / in order to 的目的句**。

角色设定：AI 演公交司机 / 地铁站工作人员 / 路人，Frank 演刚到美国的学生：
1. **问路**：Excuse me, how can I get to the train station? （听懂 AI 给的指路后复述：I turn left **to get to** Main Street, right?——练 opposite：It's opposite the bank.）
2. **上车**：问票价、刷卡方式；AI 提醒 Buckle up（系安全带），Frank 回应并说目的句：I always wear a seat belt **to stay safe**.
3. **下错站补救**：Frank 说明自己坐过站了，问怎么办（I got off at the wrong station. What should I do **in order to get back**?）
4. 收尾：Frank 用 4 句话讲一遍"我今天怎么从家到学校"的路线，至少含 2 个目的状语。
