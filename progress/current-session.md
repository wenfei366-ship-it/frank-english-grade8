## 📖 使用规则（学习顺序严格 · 日历日期灵活 · 补习课节点严格）

1. **每次开课**：AI 先读 `student-profile.md` → 本文件 → 上面标注的 Session 文件，然后按 Session 内容上课
2. **每次下课**：把下面的打卡表追加一行，并更新"下一个要学的 Session"编号
3. **中断规则**：Frank 哪天没学，下次**从记录的位置继续**——不补双倍任务，也不因缺课自动跳过任何 Session
4. **补习课硬节点**：
   - 每单元的**第 4 次 Session（问题清单打包）必须在周五补习课之前完成**；来不及时优先完成"关键课文+问题清单"两项
   - **周五**有两种模式：有补习课 → 整理问题带去答疑；无补习课 → AI 做复习和模拟答疑，问题保留到周日问老师
   - **周日补习课后**：把老师反馈写入 `progress/tutor-feedback.md`，错题写入 `progress/mistake-bank.md`
