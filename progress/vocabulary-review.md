# A 级词汇复习队列（艾宾浩斯复习曲线版）

> 只有 **A 级主动词汇**进这个队列。B/C 级不做主动复习。各单元 A/B/C 分级表见 `plans/vocab-abc/unit1~6.md`。

## 复习曲线（每个 A 级词按节点复习）

**D0（学习当天）→ D1 → D3 → D7 → D14 → D30**

- 每次 Session 开始前，先处理**已到期未复习**的词
- 缺一天不需要补双倍，下次学习先补最早到期的

## 掌握等级（0-4）

| 等级 | 标准 |
|---|---|
| 0 | 完全想不起来 |
| 1 | 看到英文知道中文 |
| 2 | 中文提示能说英文 |
| 3 | 能在句子中主动使用 ← **达到 3 才算主动掌握** |
| 4 | 能在真实对话中自然使用 |

D14、D30 都达到 3-4 → 归档（不再排复习）。用错的词退回上一个复习节点，不删除。

## 每次复习方式（每词 4 步）

1. 看中文说英文 → 2. 看英文说意思 → 3. 独立造句 → 4. AI 追问一句，Frank 在回答中再用一次
